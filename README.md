### Init
## Backend
 - `cd ./backend`
 - Copy .env.dist to .env and fill the file with env variables, then run `export ```cat .env````
 - Run `npm ci`
 - Build with `npm run build`, then serve with `npm run start`

## Frontend
 - `cd ./frontend`
 - Copy .env.dist to .env and fill the file with env variables, then run `export ```cat .env````
 - Run `npm ci`
 - Build with `npm run dev`
